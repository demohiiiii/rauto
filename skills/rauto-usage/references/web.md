# Rauto Web UI Reference

Use this reference to map user requests to the correct UI locations.

## Primary tabs

- **Operations**
  - Execute: Direct or Template via the inner “Execute” tabs
  - Transaction Block / Workflow: in “Transaction Block” card
- **Interactive**
  - Start, send commands, stop session
- **Session Replay**
  - Paste JSONL, list entries, replay a command
- **Prompt Profiles**
  - Built-in view, custom create/update, diagnose
- **Template Manager**
  - List, view, edit, create/update templates

## Connection section

- **Saved Connections**: select, save, delete, history drawer
- **Connection Defaults**: host/port/user/pass/profile inputs

## Drawers / overlays

- **Recording drawer**: floating “REC” button shows recording list and filters
- **History drawer**: “History” button in saved connections
- **Detail drawer**: event details when clicking “View Detail”
